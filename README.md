# D3 + React Bar Chart Tutorial

A beginner-friendly tutorial for building interactive bar charts using D3.js and React. This project accompanies a video from [DataVizDad](https://www.youtube.com/@datavizdad).

## About

This repo contains the code from the DataVizDad tutorial video. It walks through building a bar chart from scratch using D3 for data processing and scales, and React for rendering and state management.

The project uses a dataset of major literary prize winners and judges to create a horizontal bar chart showing the top prizes by number of entries.

## What's Included

- Starter code with TODO comments to follow along with the video
- CSV data loading and parsing with D3
- Data aggregation and transformation utilities
- A reusable `BarChart` component using D3 scales and React SVG rendering
- Vite + React project setup

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19
- D3.js
- Vite
