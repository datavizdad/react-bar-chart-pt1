# D3 + React Bar Chart Tutorial

A beginner-friendly tutorial for building bar charts using D3.js and React. This project accompanies a video from [DataVizDad](https://www.youtube.com/@datavizdad).

## About

This repo contains the code from the DataVizDad tutorial video. It walks through building a vertical bar chart from scratch using D3 for scales and React for rendering SVG.

The project uses a simple hardcoded dataset to keep the focus on learning D3 scales and SVG rendering fundamentals.

## What's Included

- A reusable `BarChart` component using D3 scales and React SVG rendering
- Sample data with labeled categories and numeric values
- Vite + React project setup

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19
- D3.js
- Vite
